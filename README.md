# utils

### 檔案功能介紹

| 名稱                      | 說明                              |
| ------------------------- | --------------------------------- |
| `cuda_version.py`         | 查看 PyTorch、CUDA、Python 版本   |
| `date_time_calculator.py` | 日期計算機                        |
| `delete_ds_store.py`      | 刪除 macOS 自動生成的 `.DS_Store` |
| `one_hot.py`              | 轉換成 One Hot Encoding 格式      |
| `qr_code.py`              | 將網站轉成 QR code                |
| `timer.py`                | 計時器                            |
| `yt_download.py`          | 下載 Youtube 影片                 |
